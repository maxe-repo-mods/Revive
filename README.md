# Revive

Press F1 to revive all dead players (host only). Configurable HP, HP cost, and invincibility duration.

R.E.P.O. BepInEx mod. Host-side only.

## Install

Requires [BepInEx 5.x](https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/).

Place `Revive.dll` in `BepInEx/plugins/`.

## Configuration

Edit `BepInEx/config/maxenterme.Revive.cfg` after first launch.

## Build

```
dotnet build -c Release
```
